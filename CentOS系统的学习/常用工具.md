1.设置Haproxy代理

安装：yum install -y haproxy

编辑代理：

```shell
cd /etc/haproxy
vim haproxy.cfg
```

```cfg
#后端mysql配置，访问虚机地址+端口号可以访问后端的mysql，例如：192.168.108.133:7306可以访问140/133的mysql数据库
listen  mysql
        bind 0.0.0.0:3306
        mode tcp
        balance roundrobin
        server mysql1 192.168.108.140:3306　　　　 			server mysql2 192.168.108.133:3306
```
